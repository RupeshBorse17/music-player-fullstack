🎵 BeatStream – Full Stack Music Player

BeatStream is a full-stack music player application built using the MERN stack. It allows users to manage songs and playlists with a smooth and interactive user experience.

Features
1. Play and manage songs
2. Create and manage playlists
3.User authentication using JWT
4. Dynamic song browsing
5.Fast and responsive UI
6.Real-time updates with API integration
